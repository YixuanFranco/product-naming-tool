### 方式2：通过后端API（推荐，避免CORS）

**后端代码（Python Flask）：**

```python
from flask import Flask, request, jsonify
from flask_cors import CORS
from anthropic import Anthropic
import os
import json
import re

app = Flask(__name__)
CORS(app)

client = Anthropic(api_key=os.environ.get("ANTHROPIC_API_KEY"))

@app.route('/api/recognize', methods=['POST'])
def recognize():
    try:
        data = request.json
        images = data['images']  # 3张base64图片
        
        # 调用Claude Vision API
        message = client.messages.create(
            model="claude-sonnet-4-20250514",
            max_tokens=1000,
            messages=[{
                "role": "user",
                "content": [
                    {
                        "type": "image",
                        "source": {
                            "type": "base64",
                            "media_type": "image/jpeg",
                            "data": images[0]
                        }
                    },
                    {
                        "type": "image",
                        "source": {
                            "type": "base64",
                            "media_type": "image/jpeg",
                            "data": images[1]
                        }
                    },
                    {
                        "type": "image",
                        "source": {
                            "type": "base64",
                            "media_type": "image/jpeg",
                            "data": images[2]
                        }
                    },
                    {
                        "type": "text",
                        "text": """这是一件藏传佛教文物的照片（正面、背面、底部）。

请你做两件事：

1. 给出一个简短的商品名称
   格式：材质+年代+题材+类型
   例如：老铜六路财神托甲牌
   要求：不超过15个字，使用藏家口吻，不提及具体朝代

2. 生成一段合规的直播话术
   要求：
   - 基于藏家角度，不是艺术品鉴定
   - 强调：这是收藏爱好者的个人藏品
   - 不说"清代""民国"等具体朝代
   - 不做功效宣传
   - 不夸大年代和价值
   - 话术长度150-200字
   - 口语化，适合直播讲解
   - 必须提及"支持7天无理由退换"

请按以下JSON格式返回：
{
  "productName": "商品名称",
  "liveScript": "直播话术内容"
}"""
                    }
                ]
            }]
        )
        
        response_text = message.content[0].text
        
        # 解析JSON
        try:
            # 移除可能的markdown代码块
            json_text = re.sub(r'```json\n?', '', response_text)
            json_text = re.sub(r'```\n?', '', json_text).strip()
            result = json.loads(json_text)
            
            return jsonify({
                'success': True,
                'productName': result['productName'],
                'liveScript': result['liveScript']
            })
        except json.JSONDecodeError:
            # 如果解析失败，尝试手动提取
            return jsonify({
                'success': True,
                'productName': response_text[:50],
                'liveScript': response_text
            })
        
    except Exception as e:
        return jsonify({
            'success': False,
            'error': str(e)
        }), 500

if __name__ == '__main__':
    app.run(host='0.0.0.0', port=5000, debug=True)
```

**前端调用（HTML中修改第307行）：**

```javascript
async function callClaudeAPI(images) {
    const response = await fetch('http://localhost:5000/api/recognize', {
        method: 'POST',
        headers: {
            'Content-Type': 'application/json',
        },
        body: JSON.stringify({
            images: images
        })
    });
    
    const data = await response.json();
    
    if (!data.success) {
        throw new Error(data.error);
    }
    
    return {
        productName: data.productName,
        liveScript: data.liveScript
    };
}

## 📋 使用流程

### Step 1: 上传图片
1. 点击5个框，分别上传：正面、背面、底部、克重、长度
2. 或者批量选择5张图片

### Step 2: AI识别
1. 上传前3张图后，"AI智能识别"按钮会启用
2. 点击按钮，等待5-10秒
3. AI会返回：
   - ✅ 商品名称
   - ✅ 直播话术

### Step 3: 确认编辑
1. 检查AI生成的名称，可以手动修改
2. 检查直播话术，可以复制使用
3. 确认商品编号（自动递增）

### Step 4: 下载
1. 点击"📦 打包下载ZIP文件"
2. 得到一个ZIP文件，包含：
   - 5张重命名的图片
   - 1个直播话术文本文件

### Step 5: 继续处理
1. 点击"🔄 继续处理下一件商品"
2. 编号自动+1
3. 所有内容清空，可以处理下一件

## 🚨 注意事项

### API调用成本
- 每次识别调用一次Claude API
- 包含3张图片 + 文本prompt
- 建议使用Sonnet模型（性价比高）

### 话术合规性
- AI生成的话术已经过合规训练
- 但仍需人工检查
- 确保没有违规词汇

### 图片格式
- 支持HEIC、JPG、PNG等所有常见格式
- 保留原始扩展名
- 建议图片大小<5MB
