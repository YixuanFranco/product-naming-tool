# 🏺 商品拍照命名助手 (Product Naming Tool) v2.0

这是一个专为电商、藏家、拍卖从业者设计的纯前端工具，旨在通过 AI 智能识别商品照片，自动生成合规的商品名称和直播/销售话术，并一键打包下载重命名后的图片。

## 🌟 核心功能

*   **AI 智能识别**：基于火山方舟（豆包/Doubao）多模态大模型，自动分析商品正面、背面、底部细节。
*   **合规命名**：生成符合藏家口吻、避开违规词汇的商品名称（材质+工艺+题材+类型）。
*   **直播话术**：自动生成 150-200 字的专业直播话术，支持 7 天无理由退换引导。
*   **打包下载**：一键生成 ZIP 包，包含：
    *   按照统一编号规则命名的 5 张商品照片。
    *   自动生成的直播话术文本文件 (`.txt`)。
*   **自动编号**：支持编号自动递增，适合批量处理。

## 🚀 部署与使用

### 在线体验
您可以直接通过本项目地址在线使用（由于纯前端运行，建议安装 CORS 插件以获得最佳 AI 体验）。

### 本地运行
1.  下载本项目。
2.  使用 VSCode 的 `Live Server` 插件运行 `index.html`。
3.  在页面顶部「API 配置」中填入您的火山方舟 API Key 和端点 ID。

## 🔑 配置说明
本工具使用**火山引擎方舟 (Volcengine Ark)** API。
- **API Key**: 从火山引擎控制台获取。
- **Endpoint ID**: 您的推理接入点 ID。

## 🛡️ 安全提示
*   本工具为纯前端实现，您的 API 配置仅保存在浏览器本地 `localStorage` 中，不会上传到任何服务器。
*   在 GitHub Pages 上运行代码是公开的，但由于 API Key 是您手动在运行界面输入的，因此非常安全。
